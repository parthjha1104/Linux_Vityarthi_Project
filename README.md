# 🧾 Open Source Audit Report – MySQL

## 📌 Overview
This project is a **comprehensive audit of MySQL**, conducted as part of the Open Source Software (NGMC Course). It explores the **technical architecture, licensing model, ethical implications, and ecosystem impact** of one of the most widely used open-source databases.

The report goes beyond surface-level documentation and provides a **deep analytical perspective** on how MySQL operates within the modern computing landscape.

---

## 🎯 Objectives

- Analyze the **origin, evolution, and purpose** of MySQL  
- Examine the **GPL licensing model and dual licensing strategy**  
- Understand **Linux system integration and architecture**  
- Evaluate MySQL’s role in the **FOSS ecosystem**  
- Compare MySQL with **proprietary database systems**  
- Implement **practical shell scripts for system auditing**  

---

## 🧠 Key Concepts Covered

- Open Source Philosophy & Ethics  
- GNU General Public License (GPLv2)  
- Dual Licensing Model  
- Linux File System & Permissions  
- systemd Service Management  
- LAMP Stack Architecture  
- Community Forking (MariaDB)  
- Security & Least Privilege Principle

## 🔐 Key Insights

- MySQL balances **open-source freedom and enterprise monetization** through dual licensing  
- Strong **Linux integration ensures performance and security**  
- The **GPL license protects user freedom** but creates business constraints  
- **MariaDB fork demonstrates the power of community governance**  
- Open source thrives on **collaboration but faces ethical challenges with corporations**
